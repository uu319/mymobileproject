This is my Mobile Project using Java programming and SQLite as database.

This is a lending management system.

The system has 3 users, the admin, staff and customer.

*Admin can:
-Admin account(username: 101 password: admin)
1. Add staff.
2. View all persons(staff and customers)
3. View all Loans
4. View all payments
5. View reports
6. View passed due loans.

*Staff can:
-Staff account(username: person ID during registration password: password provided by the admin)
1. Add customer
2. Add loans to customer
3. View persons(staff and customer)
4. Display all loans
5. Add payment
6. View all payments

*Customer can:
-Staff account(username: person ID during registration password: password provided by the admin)
1. View payments
2. View attendance
